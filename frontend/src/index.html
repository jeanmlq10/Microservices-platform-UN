Plataforma de Microservicios Dinámicos

Crea, administra y elimina microservicios en tiempo real desde el navegador. Cada servicio corre en su propio contenedor Docker y se expone automáticamente como endpoint HTTP.

¿Qué hace esta plataforma?

1
Abre el dashboard

Accede desde cualquier navegador en http://localhost

→
2
Pega tu código

Python o Node.js — solo escribe una función process(data)

→
3
La plataforma construye

Genera el Dockerfile, construye la imagen y levanta el contenedor

→
4
Endpoint disponible

Tu microservicio responde en /services/tu-nombre

Arquitectura del Sistema

🔀
Nginx Reverse Proxy :80

Punto de entrada único. Enruta tráfico al frontend, backend y microservicios dinámicos.

🎨
Frontend Dashboard Web :3000

HTML + CSS + JS Vanilla. Sin dependencias de build. Ligero y rápido.

⚙️
Backend API REST Flask :5000

Gestiona el ciclo de vida de los microservicios usando Docker SDK for Python.

🐳
Docker Engine Contenedores Dinámicos :9001+

Cada microservicio corre en su propio contenedor con puerto asignado automáticamente.

Stack Tecnológico

🔀
Nginx

Reverse proxy principal, enruta dinámicamente a los microservicios

🐍
Python 3.12 + Flask

Backend API — Docker SDK para Python es maduro y fácil de usar

🌐
HTML5 + CSS3 + JS Vanilla

Frontend sin dependencias de build, simple y ligero

🐳
Docker + Docker Compose

Todo levanta con un solo comando: docker-compose up --build

🔧
Docker Socket Mount

Más eficiente que DinD — sin daemon adicional, imágenes compartidas con el host
